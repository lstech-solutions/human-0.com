# Política de Privacidade

Última atualização: 28 de novembro de 2025

## Introdução

A HUMΛN-Ø ("nós", "nosso" ou "nos") está comprometida em proteger sua privacidade. Esta Política de Privacidade explica como coletamos, usamos e protegemos suas informações quando você usa nossa plataforma de rastreamento de impacto Web3.

## Informações que Coletamos

### Dados Pessoais
- **Nome e informações de contato**: Quando você se registra ou entra em contato conosco
- **Dados de localização**: Quando você concorda com o rastreamento de localização para fins de impacto
- **Endereços de carteira**: Para transações Web3 e recompensas

### Dados de Uso
- **Dados de interação**: Como você usa nossa plataforma
- **Informações do dispositivo**: Tipo de dispositivo, sistema operacional, navegador
- **Dados analíticos**: Padrões de uso para melhorar nossos serviços

## Como Usamos Suas Informações

- **Prestação de serviços**: Para fornecer e melhorar nossa plataforma
- **Comunicação**: Para enviar atualizações e informações importantes
- **Análise**: Para melhorar nossos serviços e desenvolver novos recursos
- **Segurança**: Para proteger nossa plataforma e usuários

## Compartilhamento de Dados

Não vendemos suas informações pessoais a terceiros. Compartilhamos dados apenas:
- **Com provedores de serviços**: Para operar nossos serviços (ex: provedores de nuvem)
- **Por razões legais**: Quando exigido por lei
- **Com seu consentimento**: Quando você concorda explicitamente

## Segurança de Dados

Implementamos medidas técnicas e organizacionais adequadas para proteger seus dados, incluindo:
- **Criptografia**: Transmissão e armazenamento de dados são criptografados
- **Controle de acesso**: Apenas funcionários autorizados podem acessar dados
- **Revisões regulares**: Revisamos regularmente nossas medidas de segurança

## Seus Direitos

Você tem o direito de:
- **Acesso**: Ver suas informações pessoais
- **Correção**: Corrigir imprecisões
- **Exclusão**: Excluir seus dados (quando legalmente permitido)
- **Oposição**: Opor-se a certos processamentos de dados

## Transferências Internacionais de Dados

Seus dados podem ser transferidos para países onde operamos serviços. Garantimos proteção adequada de acordo com as leis aplicáveis.

## Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Mudanças significativas serão comunicadas por e-mail ou notificações proeminentes em nossa plataforma.
