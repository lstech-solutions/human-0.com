# 隐私政策

最后更新：2025年11月28日

## 简介

HUMΛN-Ø（"我们"、"我们的"或"我们"）致力于保护您的隐私。本隐私政策解释了当您使用我们的Web3影响追踪平台时，我们如何收集、使用和保护您的信息。

## 我们收集的信息

### 个人数据
- **姓名和联系信息**：当您注册或联系我们时
- **位置数据**：当您同意出于影响目的进行位置追踪时
- **钱包地址**：用于Web3交易和奖励

### 使用数据
- **交互数据**：您如何使用我们的平台
- **设备信息**：设备类型、操作系统、浏览器
- **分析数据**：用于改进我们服务的使用模式

## 我们如何使用您的信息

- **服务提供**：提供和改进我们的平台
- **沟通**：发送更新和重要信息
- **分析**：改进我们的服务并开发新功能
- **安全**：保护我们的平台和用户

## 数据共享

我们不会将您的个人信息出售给第三方。我们仅在以下情况下共享数据：
- **与服务提供商**：运营我们的服务（例如：云提供商）
- **法律原因**：法律要求时
- **经您同意**：当您明确同意时

## 数据安全

我们实施适当的技术和组织措施来保护您的数据，包括：
- **加密**：数据传输和存储都是加密的
- **访问控制**：只有授权员工可以访问数据
- **定期审查**：我们定期审查我们的安全措施

## 您的权利

您有权：
- **访问**：查看您的个人信息
- **更正**：纠正不准确之处
- **删除**：删除您的数据（法律允许时）
- **反对**：反对某些数据处理

## 国际数据传输

您的数据可能会被传输到我们运营服务的国家。我们确保根据适用法律提供适当的保护。

## 政策变更

我们可能会定期更新本隐私政策。重大变更将通过电子邮件或我们平台上的显著通知通知您。
